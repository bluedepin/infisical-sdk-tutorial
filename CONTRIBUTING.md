# Contributing

Corrections, clarifications, and improvements are welcome. Open an issue describing what you found, or submit a pull request with the fix directly. For substantial changes, please open an issue first to discuss the approach.

## What's especially welcome

- Errata in any of the code blocks (a wrong version pin, a missing flag, a deprecated API).
- Steps that didn't work on your environment, with details: OS, Docker version, what you ran, what happened.
- Clearer wording for any paragraph that read as ambiguous on first try.

## What's out of scope

- Adding new features beyond what the tutorial demonstrates. The tutorial is intentionally narrow.
- Restructuring the post into a different format. The atomic-answer-paragraph + Verify-line pattern is deliberate for both human readability and AI-agent runnability.
- Switching the licence terms.
